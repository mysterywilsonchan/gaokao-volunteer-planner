# gaokao-volunteer-planner

🎯 一个面向中国高考志愿填报的 Codex Skill，用来生成**有数据来源支撑**的志愿方案、院校专业组清单和可视化网页。

它关注的不只是“能上哪所学校”，而是帮助用户看清：

- 🏫 哪些院校专业组适合冲、稳、保
- 📊 往年分数、位次、招生人数和专业线是否匹配
- 🧩 专业组内所有专业是否都能接受，避免调剂风险
- 💰 学费、校区、中外合作和身体限制等关键成本
- 🚀 每个专业未来就业方向和推荐考研方向

## ✨ 核心亮点

- **按位次规划**：优先使用位次而不是单纯分数，构建冲、稳、保梯度。
- **专业组视角**：完整列出院校专业组内所有专业，辅助判断调剂风险。
- **数据可追溯**：优先核验省考试院、高校招生官网、阳光高考等权威来源。
- **字段完整**：覆盖招生计划、录取人数、专业线、位次、学费、来源和核验状态。
- **职业参考**：为每个专业补充热门就业方向和推荐考研方向。
- **可视化交付**：生成适配电脑和手机端的志愿填报工作台网页。

## 🧑‍🎓 用户需要提供

必需信息：

- 考生省份/地区
- 高考年份和批次
- 选科或考试科目
- 高考分数
- 位次；如未提供，应结合官方一分一段表换算

建议补充：

- 兴趣和不喜欢的学科
- 预期专业方向
- 学校优先还是专业优先
- 冲稳保风险偏好
- 地域偏好
- 学费预算和是否接受中外合作
- 就业、考研、考公等未来规划
- 是否接受组内非目标专业和调剂
- 身体条件限制

## 📌 输出内容

Skill 生成的方案或网页应包含：

- 考生画像：省份、选科、分数、位次、批次线
- 冲稳保分层：目标位次区间、分数差、位次差、风险提示
- 院校信息：学校代码、标签、省份、城市、推荐理由
- 专业组信息：专业组号、计划数、投档最低分、最低位次
- 专业明细：全部专业、招生人数、录取人数、专业线、位次、学费、来源
- 职业规划：热门就业方向、推荐考研方向、专业建议
- 核验摘要：已核验字段、待核验字段和数据来源说明

## 🔍 数据原则

优先级：

1. 省级教育考试院
2. 高校本科招生官网
3. 阳光高考或教育部、高校官方平台
4. 可靠教育数据库，用作补充或交叉核验
5. PDF、图片、表格资料可通过解析或 OCR 补充

原则：

- 不编造招生人数、专业线、位次、学费或专业组号。
- 无法核验的字段必须标记为“待核验”。
- 用户要求补齐数据时，应继续查找权威资料，而不是只附链接。
- 宁可少推荐，也不要推荐数据不完整且未说明风险的专业组。

## 🖥️ 网页体验

生成页面应是可直接使用的志愿填报工作台：

- 桌面端展示筛选器、总览指标、学校卡片和专业组表格
- 手机端使用单列布局，长专业名、来源链接和就业/考研信息自然换行
- 专业组详情默认展开或展开后布局稳定
- 不依赖鼠标悬停，触屏也能完整使用
- 交付前检查电脑端和手机端，避免文字重叠、按钮裁切或信息隐藏

## ✅ 质量底线

- 每个学校有省份和城市
- 每个专业组有专业组号
- 每个专业组列全组内所有专业
- 每个专业尽量补齐招生人数、录取人数、专业线、位次和学费
- 每个专业都有就业方向和考研方向
- 最终填报前必须以省级志愿填报辅助系统和最新招生专业目录为准
